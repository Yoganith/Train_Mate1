## 🚀 How to Use

### **Seed the Database**
```powershell
cd backend
node seed-trains.js
```

### **Search for Trains**
```powershell
# Example searches
Invoke-RestMethod -Uri "http://localhost:5000/api/trains/search?from=delhi&to=mumbai"
Invoke-RestMethod -Uri "http://localhost:5000/api/trains/search?from=bangalore&to=delhi"
Invoke-RestMethod -Uri "http://localhost:5000/api/trains/search?from=chennai&to=delhi"
```

### **View in MongoDB Compass**
1. Connect to: `mongodb://localhost:27017`
2. Database: `train_booking`
3. Collection: `trains` (46 documents)
